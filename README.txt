קבצים:
- index.html — קובץ אתר מוכן לפריסה (חד-קובייתי: כולל תמונה מוטמעת)

איך מעלים לאוויר (אפשר אחת מהדרכים):
1) Netlify (הכי מהיר)
   - netlify.com → Add new site → Deploy manually
   - לגרור את index.html לחלון → מקבלים כתובת מיידית (your-site.netlify.app)
   - לחיבור דומיין: Settings → Domain management → Add custom domain → לעדכן רשומות DNS לפי ההנחיות

2) GitHub Pages
   - לפתוח ריפו חדש, להעלות index.html לענף main
   - Settings → Pages → Source: Deploy from a branch → Branch: main → Save
   - מתקבלת כתובת ציבורית. לחיבור דומיין: "Custom domain" ולהגדיר DNS

3) שרת קיים / cPanel
   - cPanel → File Manager → public_html (או תיקיית הדומיין/תת-דומיין)
   - להעלות index.html ולוודא שהוא נקרא בדיוק "index.html"

- כפתור "שיחה" (tel:) עובד מצוין בנייד.
- כפתור "מייל" (mailto:) פותח מייל חדש עם נושא וטקסט פתיחה.
